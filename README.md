# 踩地雷 3D 版

這是敝人在 2026 Spring 在國立臺灣師範大學程式設計(二) 的期末專題，使用 Claude 進行 Vibe coding 的產物

## 遊戲簡介

傳統的踩地雷是在二維的地圖上進行，而這個專案是基於相同的玩法，在三維的地圖上進行。

## 遊戲介面說明
### 設定介面
- 參數設定
  - Width 地圖寬度 (2D面上的寬)
  - Height 地圖高度 (2D面上的高)
  - Depth 地圖深度 (有幾層)
  - Mines 有多少炸彈 (1 ~ 地圖數量 - 1)
- 地圖預覽
  - Board 地圖尺寸
  - Safe 非炸彈的格子數量
  - Density 炸彈密度
### 遊戲介面
- 2D 地圖
  - 格子
    - 數字
    - 空格 (.)
    - 炸彈 (*)
    - 未翻開 (#)
- 3D 全景
- 操作指示
